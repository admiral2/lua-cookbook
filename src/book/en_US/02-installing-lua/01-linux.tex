\subsection{Installing from source}

You'll need \verb!libreadline!.

\begin{CONSOLE}
wget http://www.lua.org/ftp/lua--5.1.4.tar.gz
tar -zxf lua--5.1.4.tar.gz
cd lua--5.1.4.tar.gz
make linux
sudo make install
\end{CONSOLE}
